# ファクトチェック（fact-check）

## 検証しやすい部分

| 項目 | このアプリでの扱い | メモ |
|------|-------------------|------|
| 成人男性・女性の平均身長のオーダー | 男性 171 cm・女性 158 cm を代表値として使用 | 公的統計の概算レンジと整合しやすいが、**調査年・定義で変動**する。表示はあくまでフェルミ用途。 |
| 正規分布近似 | 身長・顔スコアに適用 | 分布の歪み（切片・選別）は無視。 |

参考として、[e-Stat](https://www.e-stat.go.jp/) や厚生労働省の「国民健康・栄養調査」概要などで身長の記述を確認できる（リンクは調査版によってパスが変わるため、ここでは固定 URL を切らない）。

## 検証が困難／主観が支配的な部分

- **顔面偏差値の人口分布**を N(50, 10) としたこと。
- **体格・運動神経・声・オーラ**の離散比。
- **加齢補正**の係数。

これらは再現性のある推定ではなく、`docs/assumptions.md` に用途を限定して記載している。

## 故意にやっていないこと

- 整形・トレーニング・护肤など**後天要因の分離**。
- 地域・サブカルチャー別の選好。
- 「成功率」や「モテ」の定義付け。

更新するときは、`src/data/assumptions.ts` の `note` と本文ドキュメントをセットで直すと、UI の説明と齟齬が出にくい。
