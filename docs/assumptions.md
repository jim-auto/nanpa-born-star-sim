# モデル仮定（assumptions）

このアプリは**娯楽・思考実験用**のフェルミ推定です。因子間の相関・努力・文脈・文化差は基本的に無視します。

## かけ算スキーム（フェルミ）

1. ベースの割合を `1.0` とする。
2. オンにした各モジュールについて比 `ratio ∈ (0, 1]` を**順に掛ける**。
3. 最終的な積を `P` とする（画面の％やレア度はこの `P` をそのまま表示）。
4. 見出しの「生まれた星偏差値」だけ別定義（次節）。

## 入力 UI

- **顔・身長・体格・運動・声・IQ**: **35–85 の偏差値スライダー**。
- **年代感・実家の太さ・育ちの地域**: **プルダウン**（係数 `ratio`。人口の尾確率ではない）。
- **10代後半**は年代の先頭にあり、係数は 20 代前半と同じ 1.0 に正規化（細かい差は捨てる）。

### A. 尾モデル（顔・体格・運動・声）

各次元で母集団スコアを **平均 50・標準偏差 10 の正規分布**とみなし、`ratio = P(X ≥ D)`。

### B. 身長

`z = (D − 50) / 10` を性別別の身長正規近似（男性 171±7 cm、女性 158±6 cm）に写像し、同じ z で尾確率。

### C. IQ

スライダー D から **目安 IQ** を `100 + 15 × (D − 50) / 10` とし、人口 IQ を **平均 100・SD 15 の正規分布**とみなして `P(IQ ≥ 目安)` を尾確率とする。実検査・WAIS 等とは無関係のフェルミ。

### D. 年代・実家・地域

`sceneAgeOptions` / `familyWealthOptions` / `birthRegionOptions` の **`ratio` を係数として掛ける**だけ。

## 生まれた星偏差値の定義（出力）

各モジュールの「比」を掛け合わせてできた**最終の積**を `P`、**オンにしたモジュール数**を `n` とする（ベースステップは含めない）。

表示ではまず `q = P^(1/n)` とし、

`偏差値風スコア = 50 + 10 × Φ⁻¹(1 − q)`

を使う（極端値は数値的にクリップ）。

- **レア度ラベル・％表示は生の積 `P`** に基づく。
- 学術指標ではなく**アプリ内スケール**。

## 独立同時の限界

実際には各次元は相関しうる。独立に掛け合わせると端の組み合わせで**過小または過大**になりうる。意図的に単純化している。
